              Centos虚拟机扩展磁盘

              简介 #

              CentOS7虚拟机原硬盘空间只分配了10GB，需要扩容到20GB。 环境：VMware 10

              VMware分配空间 #

              选中虚拟机->虚拟机设置->硬盘->实用工具->扩展->设置最大磁盘大小->点击扩展

              CentOS7内部分配 #

              可以参考：http://jingyan.baidu.com/article/54b6b9c0fc8b0b2d583b47c6.html
